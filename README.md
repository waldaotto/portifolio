## 📖 Sobre o projeto

Este projeto foi criado com o objetivo de servir como meu portfólio profissional, reunindo informações sobre mim, tecnologias que utilizo, projetos desenvolvidos e formas de contato.

O foco principal do projeto é oferecer uma interface moderna, responsiva e objetiva.

---

## ✨ Funcionalidades

- Layout moderno e responsivo
- Navegação suave entre seções
- Sessão de conhecimentos
- Sessão de projetos
- Sessão de contatos
- Efeitos visuais e transições CSS
- Compatibilidade com dispositivos móveis
